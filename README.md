# firebase-ai-logic-android-demo

Firebase AI Logic Android Demo for GDG DevFest 2025, Android Workshop
> Building an AI-Powered Product Scanner with Android & Firebase AI Logic

## Requirements
- Git
- JDK 11
- Android Studio Otter + SDK
- Android Emulator or Physical Device

## Setup Firebase Project
1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project
3. Add an Android app to the project
4. Register the app with your app's package name (e.g., `id.yuana.product.scanner.app`)
5. Download the `google-services.json` file and place it in the `app/` directory of your Android project
6. Enable Firebase AI Logic in the Firebase Console
7. Create and configure your AI Logic models as needed
8. Integrate Firebase AI Logic SDK into your Android project by adding the necessary dependencies in your `build.gradle` files
9. Initialize Firebase in your Android app
10. Test the integration by running your app and ensuring it can communicate with Firebase AI Logic

## How to Run
1. Clone this repository
2. Open the project in Android Studio
3. Sync the project with Gradle files
4. Connect your Android device or start an emulator
5. Run the app from Android Studio
